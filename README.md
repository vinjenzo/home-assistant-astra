# Home Assistant Automations & Integrations

A collection of Home Assistant automations, scripts, and custom integrations used to manage lighting, sensors, energy monitoring, and daily routines.

## Highlights
- Modular, event-driven automations
- Custom scripts for lighting and presence detection
- Voice assistant integrations
- Dashboard configuration (Lovelace)
- YAML best practices and maintainability

## Structure
- /automations – event-driven workflows
- /scripts – reusable logic blocks
- /packages – grouped domain configurations
- /custom_sentences – voice assistant intents

## Purpose
This project serves as a real-world playground for experimenting with architecture, automation patterns, and system reliability.
